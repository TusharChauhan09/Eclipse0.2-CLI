## ğŸš€ Setup & Installation

### Prerequisites

- Node.js 18+
- PostgreSQL database (local Docker or cloud like Neon)
- GitHub OAuth App (for authentication)
- Google AI API Key (for Gemini)

### Step 1: Clone & Install

```bash
cd server
npm install
```

### Step 2: Configure Environment

Create a `.env` file:

```env
# Server
PORT=3001

# Database (Local Docker example)
DATABASE_URL="postgresql://postgres:postgres@localhost:5432/ecldb"

# Better Auth
BETTER_AUTH_SECRET=your-secret-key-here
BETTER_AUTH_URL=http://localhost:3001

# GitHub OAuth
GITHUB_CLIENT_ID=your-github-client-id
GITHUB_CLIENT_SECRET=your-github-client-secret

# Google AI
GOOGLE_GENERATIVE_AI_API_KEY=your-google-ai-api-key
ECLIPSE_MODEL=gemini-2.5-flash
```

### Step 3: Setup Database

```bash
# Start local PostgreSQL (Docker)
docker run --name ecl-postgres -e POSTGRES_USER=postgres -e POSTGRES_PASSWORD=postgres -e POSTGRES_DB=ecldb -p 5432:5432 -d postgres:16-alpine

# Run migrations
npx prisma migrate deploy

# Generate Prisma client
npx prisma generate
```

### Step 4: Build & Run

```bash
# Build TypeScript
npm run build

# Start server
npm run dev
```
